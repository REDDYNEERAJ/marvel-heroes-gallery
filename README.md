# Marvel Heroes Gallery

## Overview
Marvel Heroes Gallery is a Django web application that showcases Marvel superheroes, providing detailed profiles and information for each character. Built with Django, Python, HTML, and CSS, it offers fans an interactive way to explore the Avengers universe.

## Features
- Browse Marvel superheroes
- Detailed character profiles
- Interactive and responsive UI
- Built with Django framework

## Technologies Used
- Django
- Python
- HTML, CSS
- SQLite (default Django database)
